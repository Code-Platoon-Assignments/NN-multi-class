# Multi-Classification

## Instructions

In this exercise you will create 3 separate Multi-Classification models that will utilize the provided data for training.

- [Air Quality and Health](https://www.kaggle.com/datasets/rabieelkharoua/air-quality-and-health-impact-dataset)
- [Online Gaming](https://www.kaggle.com/datasets/rabieelkharoua/predict-online-gaming-behavior-dataset)

### Tools you may need

- [ReLu](https://pytorch.org/docs/stable/generated/torch.nn.ReLU.html)
- [Leaky ReLu](https://pytorch.org/docs/stable/generated/torch.nn.LeakyReLU.html)
- [Step Scheduler](https://pytorch.org/docs/stable/generated/torch.optim.lr_scheduler.StepLR.html)
- [Label Encoding](https://scikit-learn.org/stable/modules/generated/sklearn.preprocessing.LabelEncoder.html)
